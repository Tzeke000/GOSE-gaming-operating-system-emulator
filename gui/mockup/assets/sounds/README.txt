  nav.wav     — moving focus (arrows / d-pad)
  select.wav  — confirm / A / Enter
  back.wav    — back / B / Esc
  launch.wav  — launching a game/app
  error.wav   — failed action
Short (<200ms), mono or stereo WAV. Toggle in Settings → Sound → UI sounds.
