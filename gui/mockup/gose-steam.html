PROTON & WINE

Proton (via Wine-GE + DXVK)
Batocera ships Wine-GE + DXVK in the windows system. Steam uses this path for Windows games. Light and older titles that don’t need Vulkan will run.
GPU ceiling (VM)
This VM uses virgl (OpenGL 4.3, no Vulkan). DXVK needs Vulkan for modern DirectX 11/12 games. Modern AAA titles won’t run here — they need real hardware.
Anti-cheat
Games using kernel-level anti-cheat (EAC, BattlEye) won’t work under Proton on any Linux VM — this is a vendor restriction, not a GOSE limitation.
Full GPU — Vulkan available
This device has Vulkan. Most Steam/Proton titles including DirectX 11/12 games should work. Performance depends on hardware.
