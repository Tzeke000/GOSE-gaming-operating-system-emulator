HOW TO CONNECT
1
Enable file sharing above.
2
Windows: open File Explorer, click the address bar, type the address shown above and press Enter. Or press Win+R, paste the address and press Enter.
3
Mac: open Finder, press Cmd+K, type smb://<ip-address> and click Connect.
4
Drop ROM files into the folder for each system. They appear in GOSE after a library scan (Library › Scan).
5
Disable file sharing when done — keeping it on exposes your device to your local network.
