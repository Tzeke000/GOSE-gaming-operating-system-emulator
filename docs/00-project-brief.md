# Odin 2 Custom Gaming OS — Original Project Brief (preserved)

> truth for intent. Verified facts and any corrections live in

**Owner:** Zeke (Tzeke000)
**Prepared for:** Claude Code (and future me)
**Goal in one line:** Turn an AYN Odin 2 into a console-like, controller-driven gaming + tinkering device running a flashable Linux OS, with a Windows-style GUI, broad emulation, universal controller support, and the ability to talk to my AI agents.

## 0. Scope & honest framing
Not "write an OS from scratch." It IS: take a mature handheld Linux gaming OS
(Batocera or ROCKNIX), flash it to SD, and configure + customize it to meet every
requirement — including a custom Windows-like front-end, controller mappings, and
network hooks to my AI agents. Custom-built pieces (home UI, AI bridge) are
flagged `[CUSTOM]`. Must be fully driveable with the Odin 2's native controls — no
keyboard required for normal use.

## 1. Hardware target
AYN Odin 2 (Snapdragon 8 Gen 2, Adreno 740, 8GB+ RAM, 6" 1080p, Wi-Fi 6/7 + BT
5.x, 1× USB-C OTG, 3.5mm, microSD, ~8000mAh). Boot custom OS from SD, keep Android
on internal. Expansion via AYN Super Dock (GbE, HDMI, USB 3.0/2.0, NVMe), portable
USB-C OTG hub with PD pass-through, 8BitDo USB Wireless Adapter 2.

## 2. OS candidates
Batocera (recommended default, biggest coverage) or ROCKNIX (clean, good Odin
support). Verify support for the EXACT model/SoC before flashing. Caveat: dock
HDMI may fail on Linux; keep USB-C→HDMI adapter as fallback.

## 3. Functional requirements
- **Emulation:** PSP (flagship, must run great upscaled), Switch (best-effort),
  NES/SNES/GB/GBC/GBA/DS, Genesis, PS1, PS2, GameCube, N64, Dreamcast, arcade,
  light PC via translation (Box64/Wine/Winlator-style, best-effort). Each system
  pre-configured with sane defaults; user never edits configs by hand.
- **Controllers:** Xbox (xpadneo, BT+USB), PS4/PS5 (native), Switch Pro/Joy-Cons
  (fussy), 8BitDo (native), universal via 8BitDo USB Adapter 2. Multiplayer up to
  ~6 (BT handles 3–4; rest via dock USB when docked to TV).
- **Peripherals:** HDMI out (dock or USB-C→HDMI), USB-C OTG host, USB-A via hub,
  Bluetooth, Ethernet (dock) + Wi-Fi, microSD as "cartridges," USB mic fallback.
- **Keyboard-optional rule:** normal use = controller only; on-screen keyboard for
  rare text entry.

## 4. The GUI [CUSTOM]
Home screen that looks/behaves like a Windows desktop but is 100%
controller-navigable. Flow: Home → pick system → that system's library → pick
title → auto-launch right emulator. Library per system: box art, titles, recent,
favorites. "Apps/Tools" area for the hacker side: terminal, file manager, network
tools, AI bridge launcher. Settings controller-driven with on-screen keyboard.
Options: customize the distro front-end with a Windows-style theme (fastest), OR
build a custom front-end app (more control, "really mine").

## 5. Hacker / tinkering layer
Full Linux underneath; SSH from Windows tower; terminal + file manager + network
tools in the Tools area; comfortable as a portable Linux node. Document every
customization so a re-flash is reproducible.

## 6. AI agent integration [CUSTOM]
Connect to my AI agents — **Ava, Wren, Iris** — over the network. They run on my
server; the Odin is a client/endpoint. Paths: Wi-Fi or dock Ethernet; optionally
USB/serial if tethered. Build a small AI bridge launchable from Tools: talks to
agents via their API/endpoint; optional voice in/out; could become an on-device
assistant/overlay. **Open item:** specify how Ava/Wren/Iris expose themselves.

## 7. Build order
1. Confirm model + verify distro support; pick distro + build. 2. Flash to fast A2
microSD; confirm boot + native controls. 3. Core emulation (PSP first). 4.
Controllers (xpadneo, dongles, multi-controller, ~6 via dock). 5. Peripherals
(HDMI fallback, hub, Ethernet, USB mic). 6. `[CUSTOM]` Windows-like GUI flow. 7.
`[CUSTOM]` Tools area. 8. `[CUSTOM]` AI bridge/agent. 9. Reproducible setup script
+ notes.

## 8. Gotchas
Single USB-C (charge+OTG share); dock HDMI may not work on Linux; newest SoC =
least mature drivers; Switch + light PC are best-effort per-title; 6 controllers
is a docked scenario; prefer USB mic; keep everything reproducible.

## 9. Shopping list
Odin 2 (+ Super Dock + case); fast A2 microSD 256GB+ (+ spare); 8BitDo USB
Wireless Adapter 2; USB-C OTG hub w/ PD pass-through; USB-C→HDMI adapter; optional
USB mic; optional BT/USB keyboard (setup only).
